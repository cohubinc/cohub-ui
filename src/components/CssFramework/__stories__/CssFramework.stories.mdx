# CssFramework

This component needs to be added to the root of your application one time/
It will add all the css variables and css utility classes needed for our component library.
The css vars and utils are availible for use in the consuming application

<Story name="CssFramework">See the docs tab</Story>

## Example usage

```jsx
<CssFramework />
```
