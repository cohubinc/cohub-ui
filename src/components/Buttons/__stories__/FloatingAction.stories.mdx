## Example usage

```jsx
<Buttons.FloatingAction
  icon="back"
  backgroundColor={Color.iconGrey}
  iconColor={Color.black}
  size={55}
  elevation={3}
  onClick={() => "..."}
/>
```

FloatingAction only excepts the **raised** and **backgroundColor** props

<Props of={Buttons.Base} />

**Also accepts all [native button attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button#Attributes) as props**
